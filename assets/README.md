1. Download the image from the Amazon CDN URL above
2. Optionally resize/optimize it for web use (recommended: 1200x630 for OG image)
3. Save it as `green-soul-jupiter-pro.jpg` in this directory

This ensures:
- Better page load performance
- No CORS or hotlinking issues
- Consistent image availability
- Control over image optimization
